BoomBox Remastered is a virtual reality rhythm game about hitting drums with drumsticks.

It originally released in 2021, but after some struggling with the publisher, it was taken offline to be released independently.
I was hired as a Technical Artist & Programmer to help rebuild the project.

For this project, I took all of the old environments and moved them into the new project with updated visuals.

I utilized Unity's ShaderGraph to create a standard Lit shader for the project, as well as a few other shaders, like my stylized water shader.

I additionally adjusted lighting, added subtle fog and smoke VFX, and made other tweaks to add to the feel of the environment.

The original game's environments only had a fixed length, so I created a system to procedurally fit chunks of the environment together as the song goes on.

It automatically hides and spawns pieces (with a nice little animation) and always ensures you start at the start piece and end at the end piece.

Since we were targetting a full framerate on standalone mobile headsets, I needed to pivot from realtime to baked lighting for the environments.

This introduced some problems with the dynamic environment chunks, but after looking into how baked lighting works in Unity, I created a system that transfers the lightmap data to the instantiated pieces.

I learned how to effectively utilize baked lighting, fixing broken UVs and balancing memory footprint with fidelity.

I was responsible for some small gameplay polish and features around the game, which required an understanding of the large codebase, and was an introduction to Dependency Injection. For these features, I had to sometimes create proprietary shaders along with script behavior to control the visuals.

During this project, I learned how to use UI Toolkit to create functional UI views. Since I was also creating this website at the time, I was able to easily understand how to create dynamic layouts and styles.

I also learned how UI Toolkit binds with the underlying scripting, creating efficient UI flows that efficiently updated elements using an event based approach.

BoomBox Remastered launched December 25th, 2025 on Meta and Steam.
