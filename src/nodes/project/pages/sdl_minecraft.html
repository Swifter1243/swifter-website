SDL Minecraft is a simple paint-like program made in SDL 2.

A short demo of the project.

It was a school project me and my friend Evan were partnered up for to learn SDL. It wasn't meant to be anything close to a real Minecraft clone, but moreso just a small project to learn how to properly architect and manage system resources.

You can left-click to add blocks to the world, and you can right click to add blocks. You can also use keys 0-9 to select blocks to place.

These blocks actually effect the world save, which you can serialize to disk, share with a friend, and load back up. This required a custom file format with a custom header and custom binary schema.

Every time the program starts up, we generate terrain using a noise algorithm for caves and terrain height. After the solid terrain is determined, grass is placed on the top block, then dirt 3 blocks under that, and then stone.

Decorations such as trees and tall grass are later placed on available blocks. This is very inspired by how Minecraft actually generates it's terrain.
